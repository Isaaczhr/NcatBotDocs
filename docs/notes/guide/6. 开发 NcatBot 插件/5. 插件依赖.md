## 声明插件依赖

使用 `BasePlugin.dependencies` 声明插件依赖, 格式见下.

```python
class MyPlugin(BasePlugin):
    name = "MyPlugin"
    version = "1.0.0"
    dependencies = {
        "OtherPlugin": ">=1.0.0",
        "Logger": ">=2.0.0"
    }
```
